### 🚀 Highlight Project

#### 🤖 Insurance AI Chatbot
> Python · LangChain · RAG · Streamlit · MySQL · Docker · AWS EC2

Non-technical insurance staff type questions in plain English and get instant,
accurate answers from a **100,000+ record** MySQL database — no SQL knowledge required.

**What makes it interesting:**
- Full **RAG pipeline** — schema context retrieved before every query for accuracy
- **Text-to-SQL** via LLM API with 10+ prompt engineering techniques applied
- **Safety guardrails** — blocks destructive queries (DROP, DELETE, UPDATE) automatically
- Fully **Dockerized** and deployed on **AWS EC2** — local dev to production in under 2 weeks
